# Modo Sandbox

Para facilitar el proceso de integración y permitirte hacer pruebas de las diferentes acciones que puedes realizar con Soyio, puedes utilizar un ambiente de prueba. El entorno de pruebas funciona exactamente igual que el de producción, sin embargo ninguna de las validaciones de identidad realizadas en este ambiente son válidas. Las transacciones realizadas en este entorno de prueba están aisladas del ambiente de producción.

## Base URL

El ambiente de sandbox se encuentra ubicado en la siguiente Base URL:

`https://sandbox.soyio.id`


## Endpoints y parámetros especiales

Existen algunos endpoints que sólo están disponibles en el ambiente de sandbox, por ende, no estarán disponibles en el ambiente de producción. Estos permiten realizar acciones para facilitar pruebas de integración, como por ejemplo, eliminar una identidad.

Algunos endpoints tendrán ciertos parámetros que permiten **simular comportamientos específicos**, como por ejemplo errores en la validación de identidad.
