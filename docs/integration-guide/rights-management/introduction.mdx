# Gestión de derechos

:::info[Información]
Por ahora, este servicio está disponible **sólo para el SDK Web**.
:::

Las leyes como la Ley Chilena de Protección de Datos Personales (Ley 21.719) o la GDPR, exigen a las empresas que le permitan a los usuarios ejercer sus derechos de protección de datos de manera sencilla y transparente.

El **módulo de gestión de derechos** de Soyio te permite cumplir con estos requisitos generando evidencia para fines de auditoría y a tus empleados les permite gestionar las solicitudes de manera centralizada. Usa este módulo para:

- **Visualizar y responder** las solicitudes de ejercicio de derechos de tus usuarios.
- **Notificar y recordar** a los responsables de tu empresa sobre los plazos de respuesta de las solicitudes.
- **Generar evidencia** de las solicitudes y las respuestas de la empresa.
- **Disponer de un formulario** especializado en tu sitio web o app para recibir las solicitudes.
- **Crear automatizaciones** y flujos de trabajo para procesar las solicitudes de manera eficiente.
- **Validar la identidad del usuario** en el formulario para asegurar la legitimidad de las solicitudes.
- **Centralizar la gestión de todas las solicitudes:** en una plataforma transversal para toda tu organización e integrada con tus sistemas internos.
