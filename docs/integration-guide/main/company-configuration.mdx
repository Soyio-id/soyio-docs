# Configuración de tu Compañía

Esta guía te ayuda a configurar tu cuenta de empresa en Soyio, incluyendo configuraciones de privacidad, políticas de datos y preferencias de integración.

## Configuración Inicial

### 1. Crear Cuenta de Empresa

```bash
# Crear nueva cuenta
curl -X POST https://api.soyio.com/v1/accounts \
  -H "Content-Type: application/json" \
  -d '{
    "companyName": "Mi Empresa S.L.",
    "legalName": "Mi Empresa Sociedad Limitada",
    "taxId": "B12345678",
    "contactEmail": "admin@miempresa.com",
    "country": "ES",
    "industry": "technology",
    "dataProtectionOfficer": {
      "name": "Juan Pérez",
      "email": "dpo@miempresa.com"
    }
  }'
```

### 2. Configurar Información Legal

```json
{
  "legalInfo": {
    "companyName": "Mi Empresa S.L.",
    "legalName": "Mi Empresa Sociedad Limitada",
    "taxId": "B12345678",

      "street": "Calle Principal 123",
```

## Configuración de Integración

### 1. Configuración de API

```json
{
  "apiConfiguration": {
    "rateLimits": {
      "requestsPerMinute": 1000,
      "requestsPerHour": 10000,
      "requestsPerDay": 100000
    },
    "webhookSettings": {
      "maxRetries": 3,
      "retryDelay": 5000,
      "timeout": 30000
    },
    "securitySettings": {
      "requireHttps": true,
      "allowedOrigins": ["https://miempresa.com"],
      "ipWhitelist": ["192.168.1.0/24"]
    }
  }
}
```

### 2. Configuración de Webhooks

```bash
# Configurar webhook principal
curl -X POST https://api.soyio.com/v1/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Webhook Principal",
    "url": "https://miempresa.com/api/webhooks/soyio",
    "events": [
      "consent.created",
      "consent.updated",
      "consent.withdrawn",
      "data.subject.request.created"
    ],
    "secret": "webhook_secret_key",
    "active": true
  }'
```

### 3. Configuración de SDK

```javascript
// Configuración del SDK
const soyioClient = new SoyioClient({
  apiKey: 'your_api_key',
  environment: 'production',
  baseUrl: 'https://api.soyio.com',
  timeout: 30000,
  retries: 3,
  webhookSecret: 'webhook_secret_key'
});
```

### 2. Gestión de Usuarios

```bash
# Invitar usuario
curl -X POST https://api.soyio.com/v1/users/invite \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "nuevo@miempresa.com",
    "role": "developer",
    "permissions": ["api:read", "api:write"]
  }'
```

## API de Configuración

### Obtener Configuración Actual

```bash
GET /api/v1/account/configuration
```

### Actualizar Configuración

```bash
PUT /api/v1/account/configuration
Content-Type: application/json

{
  "privacyPolicy": {
    "version": "1.1",
    "effectiveDate": "2024-02-01"
  }
}
```

### Obtener Configuración de Privacidad

```bash
GET /api/v1/account/privacy-settings
```

## Mejores Prácticas

### Configuración Inicial
- Completa toda la información legal requerida
- Configura políticas de privacidad claras
- Establece períodos de retención apropiados
- Define roles y permisos específicos

### Mantenimiento
- Revisa y actualiza configuraciones regularmente
- Monitorea el cumplimiento de políticas
- Mantén documentación actualizada
- Capacita al equipo en nuevas configuraciones

### Seguridad
- Usa HTTPS para todas las comunicaciones
- Implementa autenticación de dos factores
- Monitorea accesos y cambios
- Mantén copias de seguridad de configuraciones
