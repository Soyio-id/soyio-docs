# Módulo de consentimiento

## Introducción

El consentimiento es la base legal más reconocida para el tratamiento de datos personales. Las leyes como la Ley Chilena de Protección de Datos Personales (Ley 21.719) o la GDPR, exigen que este consentimiento sea previo, informado, explícito, específico y revocable, garantizando que las personas mantengan el control sobre el uso de su información.

El **módulo de consentimiento** de Soyio te permite cumplir con estos requisitos de manera sencilla y auditable. Usa este módulo para:

- **Capturar y actualizar consentimientos:** de tus usuarios a través de múltiples canales digitales (web, móvil, presencial, entre otros).
- **Integrar fácilmente el consentimiento en tus flujos de negocio:** sin fricción en la experiencia del usuario.
- **Generar evidencia del consentimiento otorgado:** de manera automática y auditable.
- **Centralizar la gestión de todos los consentimientos:** en una plataforma transversal para toda tu organización.
- **Habilitar un portal de gestión de consentimientos:** para que tus usuarios puedan visualizar, gestionar y revocar sus consentimientos.
