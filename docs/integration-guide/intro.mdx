# Guías de integración

<img src="/img/welcome-banner-guide.png" alt="welcome" width="100%"/>
<br/>
<br/>

Bienvenido a la documentación de integración de Soyio, la plataforma líder en gestión de privacidad y cumplimiento en protección de datos. Esta guía te ayudará a integrar nuestras soluciones de manera efectiva en tu empresa.

Soyio es una plataforma SaaS que facilita el cumplimiento de regulaciones de privacidad de datos como la Ley chilena de protección de datos personales (Ley 21.719) y GDPR.

Nuestra plataforma te proporciona las siguientes herramientas:

## Soluciones de gestión para la protección de datos

### [Consentimiento](./consent/introduction)

El servicio de consentimiento te permite:
- **Capturar consentimientos** de manera granular en tus procesos digitales.
- **Habilitar un portal de revocación de consentimientos** para que tus usuarios puedan visualizar y gestionar sus consentimientos.
- **Generar evidencia** del consentimiento otorgado, revocado o caducado.
- **Generar reportes** de auditoría y cumplimiento para demostrar tu cumplimiento ante el regulador.

### Gestión de solicitudes de ejercicio de derechos

El servicio de gestión de solicitudes de ejercicio de derechos te permite:
- **Habilitar un formulario de ejercicio de derechos** para que tus usuarios puedan envíar sus solicitudes de ejercicio de derechos.
- **Gestionar las solicitudes de ejercicio de derechos** de tus usuarios de manera eficiente.
- **Generar evidencia** del proceso de resolución de las solicitudes de ejercicio de derechos.
- **Generar reportes** de auditoría y cumplimiento para demostrar tu cumplimiento ante el regulador.

### [RAT AI](./rat/introduction)

El servicio de RAT AI es un agente conversacional basado en IA que te permite:
- **Automatizar el levantamiento de información** de las actividades de tratamiento de datos persona les dentro de tu organización.
- **Supervisar el avance** transversalmente a todos las áreas de tu organización.
- **Generar y mantener actualizado el registro de actividades de tratamiento de datos personales** de tu organización.
- **Aclarar dudas y proporcionar información** sobre conceptos de protección de datos personales.

## Soluciones de identificación

### Verificación de identidad

El servicio de verificación te permite:
- **Verificar la identidad de tus usuarios** de manera segura en entornos digitales.
- **Comprobar que tu usuario esta vivo y presente** en el momento de la verificación.
- **Comprobar que el documento de identidad sea válido y vigente**.
- **Comprobar que el documento de identidad le corresponde al usuario**.
- **Extraer información del documento de identidad** como nombre, apellido, fecha de nacimiento, género, etc.
- **Registrar la identidad verificada** de tu usuario en el sistema para fines de autenticación futura.
- **Crear una passkey** de forma opcional, para tu usuario pueda usarla para autenticación futura.

### [Autenticación](./authentication/introduction)

El servicio de autenticación te permite:
- **Autenticar a tus usuarios** de manera segura en entornos digitales.
- **Match biometrico** con la identidad previamente verificada para confirmar la identidad del usuario.
- **Autenticación con passkeys** para confirmar la identidad del usuario o crear una passkey si no tiene una registrada.

### [Firma de documentos](./signature/introduction)

El servicio de firma de documentos te permite:
- **Solicitar la firma de documentos electrónicamente** de manera segura directamente en tus canales digitales.
- **Generar plantillas de documentos** y completarlos con la información de tu usuario dinámicamente.
- **Envío de documentos firmados** por correo electrónico a los usuarios firmantes.
- **Consulta de documentos firmados** y evidencia de la firma a través de la API.

## Arquitectura de la Plataforma

Soyio está construido con una arquitectura moderna y escalable que incluye:

- **API REST**: Interfaz programática completa para todas las funcionalidades
- **Webhooks**: Notificaciones en tiempo real de eventos
- **SDKs**: Librerías para facilitar la integración en sistemas web y móviles
- **Dashboard de Administración**: Interfaz web para gestionar configuraciones y monitorear actividad

## Flujo de Integración Típico

1. **Obtener Credenciales**: Contacta a nuestro equipo de ventas para cotizar la solución y obtener las credenciales de API completando el [formulario de contacto](https://soyio.id/contact).
2. **Configuración Inicial**: Configura tu cuenta empresa y personaliza el sistema en base a tus necesidades.
3. **Implementación de Módulos**: Integra los módulos específicos que necesites.
4. **Pruebas en Sandbox**: Utiliza el entorno de pruebas para validar tu integración.
5. **Despliegue en Producción**: Migra a producción una vez que todo esté validado.
